## 4XX 客户端错误

* 400 Bad Request
    * 语法错误
* 401 Unauthorized
    * 认证失败，如
        * BASIC 认证
        * DIGEST 认证
* 403 Forbidden
    * 被拒绝
* 404 Not Found
    * 资源不存在
