Berkeley套接字（也作BSD套接字应用程序接口）刚开始是4.2BSD Unix操作系统（于1983发布）的一套应用程序接口。然而，由于AT&T的专利保护着UNIX，所以只有在1989年伯克利大学才能自由地发布自己的操作系统和网络库。

* Berkeley Socket
    * 4.2BSD Unix
        * 1983
        * AT&T Unix
    * 1989
        * OS
        * Network Interface Library

* Unix Domain Socket
    * IPC (Inter-process Communication)
