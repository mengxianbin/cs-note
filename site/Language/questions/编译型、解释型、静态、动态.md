## 编译型与解释型

> http://c.biancheng.net/view/4136.html

- 执行过程
    - 解释: 源码 -> 解释器 -> 操作系统 -> CPU
    - 编译: 源码 -> 编译器 -> 可执行文件 -> 操作系统 -> CPU

- 跨平台
    - 解释型: 解释器
    - 编译型: 虚拟机 (如果有)

## 静态与动态

> https://zhuanlan.zhihu.com/p/66383937

- 动态 (弱类型): 更灵活
- 静态 (强类型): 更安全, 更强大的 IDE 支持
