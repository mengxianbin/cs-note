# 循循善诱 而非 独断专行

- 分享事实经过
    - 从最少争议的事实开始

- 说出想法
    - 自己依据的事实
    - 自己得出的结论

- 征询对方观点
    - 对方依据的事实
    - 对方得出的结论

- 试探性表述
    - 区分 想法 和 事实

- 鼓励尝试
    - 创建安全感
    - 鼓励对方说出不同甚至对立的观点
