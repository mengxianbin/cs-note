## Range

* 请求报文

```http
GET /z4d4kWk.jpg HTTP/1.1
Host: i.imgur.com
Range: bytes=0-1023
```

* 响应报文
    * 206 Partial Content

```http
HTTP/1.1 206 Partial Content
Content-Range: bytes 0-1023/146515
Content-Length: 1024
...
(binary content)
```

## Accept-Ranges

* 响应报文
    * 支持范围请求与否
        * 支持：bytes
        * 不支持： none

```http
Accept-Ranges: bytes
```

## 响应状态码

* 范围请求成功
    * 206 Partial Content
* 不支持范围请求
    * 200 OK
* 范围越界
    * 416 Requested Range Not Satisfiable
