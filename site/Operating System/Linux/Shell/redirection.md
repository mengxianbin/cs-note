| -               | -                                                |
|-----------------|--------------------------------------------------|
| command > file  | output to file                                   |
| command < file  | input from file                                  |
| command >> file | append to file                                   |
| n >& m          | output and merge                                 |
| n <& m          | input and merge                                  |
| << tag          | input from what is between start tag and end tag |
