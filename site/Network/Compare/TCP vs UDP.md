[TCP vs UDP](https://mengxianbin.github.io/cs-notes/site/Network/Compare/TCP%20vs%20UDP)

* TCP
    * Connection-based
    * Stream oriented
    * Reliable
        * Sequence Number
            * Acknowledge
                * Re-send if lost
            * Reassemble if out of sequence

* UDP
    * Connectionless
    * Message (/ Datagram / Packet) oriented
    * Unreliable
        * no guarantee of
            * reaching
            * sequence keeping
        * but faster
