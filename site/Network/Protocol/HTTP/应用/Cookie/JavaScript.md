* document.cookie
    * 创建新 Cookie
    * 也可以访问非 HttpOnly 标记的 Cookie

```js
document.cookie = "yummy_cookie=choco";
document.cookie = "tasty_cookie=strawberry";
console.log(document.cookie);
```
